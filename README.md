# Khattat — خطاط

A gamified Arabic calligraphy learning app built with React + Vite.

## Deploy to Vercel
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import this repository
3. Click Deploy

The `vercel.json` config is already set up.

## Local Development
```bash
npm install
npm run dev
```
